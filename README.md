**Natural Language Processing (NLP) Projects**

This repository showcases hands-on Natural Language Processing (NLP) projects developed to build a strong foundation in text analytics, machine learning, and applied AI.
The work emphasizes clean code, reproducible experiments, and real-world NLP workflows.

**Key Skills Demonstrated**

Text preprocessing and normalization

Feature engineering (Bag of Words, TF-IDF)

Classical NLP modeling and evaluation

Exploratory data analysis on textual datasets

Python-based ML pipelines

**Technologies**

Programming: Python

Libraries: Pandas, NumPy, Scikit-learn

NLP Tools: NLTK, spaCy

Environment: Jupyter Notebook
